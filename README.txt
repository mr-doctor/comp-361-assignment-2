In order to run the program, the main class SequenceMatcher must be run with two input strings of characters, 
	or the main class SequenceMatcher can be run from the command line with the two arguments being the two
	input strings, for example 'ATACAGA GATCATTA'. The program will then print the output
	of the program.

To display the output in a test format, run the displayTests test in Tests.java.
To show performance, run the performanceTests in Test.java.
